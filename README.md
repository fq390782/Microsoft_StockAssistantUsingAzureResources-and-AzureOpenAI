## 프로젝트 개요

본 프로젝트는 클라우드 기반의 인프라 구현과 백엔드 구축으로 대규모 확장 가능한 IT 솔루션 아키텍처 제작을 위하여 구성되었습니다.

| :------------- | :--------------------------------------------------------------------------- |
| **핵심** | Cloud Infra 중심으로 Data Engeenering 및 Pipeline 구축                       |

### 기술적 특징

* 사회적 ・ 윤리적 문제에 대한 관점에 입각한 제품 설계 및 구현
* 클라우드 기반으로 Scale Up/Out, SLA 확보 (서비스별 상이)등 Provisioning의 유연함
* 비용 정책에 대한 분석으로 시장 실현성을 확보
* 빠른 기능 최소 구현으로 클라이언트 및 잠재 고객 유지
